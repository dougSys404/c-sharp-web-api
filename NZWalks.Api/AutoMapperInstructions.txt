Instalation by NuGet.

1º - Right click on "Dependences"
2º - Click on Manage NuGet Packages
3º - Browse for Automapper
4º - Select "Automapper" by Jimmy Bogard
5º - Click on ""Install"

Create a folder inside the project called Mapping

Create a class called AutoMapperProfiles
